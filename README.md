# raytracer
Ray tracer in OpenGL for sphere rendering for a course project

## Dependencies:
- OpenGL 3
- freeglut
- GLEW

## Features:
- Ray-sphere intersection
- Phong local illumination
- Shadows
- Super sampling
- Reflections
- a plane surface
